Lg type poem means that the poem doesn’t have stanzas, so the line group (lg) contains the whole poem.

Any truly END endnotes need to have in the poem title <ref type=”endnote” target=”#[xml:id]”>[what you want displayed as a superscript]</ref> instead of an actual note tag containing the note: we are not going to actually CREATE the note at the end (no “mode=’end’”) because they are already transcribed at the end of the witnesses. So what goes inside the note tag isn’t the note itself, it reflects where the notes appear in the witness.

Make sure that footnotes placed at the bottom of the page are NOT inside the stanza.
